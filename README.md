# 四年级数学 AI 自适应学习系统

面向北京海淀区四年级学生的数学学习辅助工具，基于北师大版教材，提供"测试 → 批改诊断 → 针对性练习 → 能力追踪"的完整学习闭环。

## 在线使用

直接用浏览器打开 `index.html` 即可使用，无需安装任何软件。

---

## 功能特点

### 📝 单元测试卷生成
- 支持 16 个单元（上下学期各 8 单元）的测试卷生成
- 标准模式：随机抽题组卷
- 自适应模式：根据薄弱知识点优先出题
- 标准 A4 格式，适合打印
- 包含选择题、填空题、计算题、应用题四大题型
- 符合海淀区校内考试难度标准

### ✅ 批改试卷与智能诊断
- 拍照上传试卷照片，对照输入答案
- 自动批改，实时计算总分
- 按知识点归纳错误类型，定位薄弱环节
- 生成辅导建议，方便家长针对性辅导

### 🎯 针对性练习（自适应出题）
- 基于错题知识点的变式训练
- 优先巩固薄弱知识点
- 题量可选（5/10/15 题）

### 📊 学生能力报告
- 整体掌握度评分
- 四大能力分类分布（数与运算/图形与几何/统计与概率/方程与代数）
- 强弱知识点排名
- 学习趋势追踪
- 练习历史记录

## 使用方法

### 第一步：添加学生
在页面顶部输入学生名字，点击"添加"，系统会为每个学生独立保存能力模型。

### 第二步：生成测试卷
1. 切换到"单元测试"标签
2. 选择学期和单元
3. 选择出题模式（标准/自适应）
4. 点击"生成测试卷"
5. 打印或复制试卷给孩子做

### 第三步：批改诊断
1. 做完试卷后，切换到"批改诊断"标签
2. 拍照上传试卷（或手动输入答案）
3. 对照照片逐题输入学生答案
4. 格式：`1. D` / `6. 10, 100` / `12. 5000`
5. 点击"开始批改"，查看分数、错题分析和辅导建议

### 第四步：针对性练习
1. 切换到"针对性练习"标签
2. 系统自动展示当前学生的薄弱知识点
3. 选择单元和题量
4. 点击"生成针对性练习"

### 第五步：查看能力报告
1. 切换到"能力报告"标签
2. 查看整体掌握度、能力分布、强弱项排名
3. 查看练习历史记录

## 知识图谱

系统内置完整的四年级数学知识图谱，共约 60 个细粒度知识点，覆盖四大能力分类：

- **数与运算**：万以内数的读写、数位与计数单位、大小比较、近似估算、大数运算、乘法、除法、小数、负数等
- **图形与几何**：线与角、轴对称、平移旋转、方向位置、三角形四边形、观察物体等
- **统计与概率**：统计图、平均数、可能性等
- **方程与代数**：方程概念、解方程、列方程解应用题

知识点之间标注了依赖关系，诊断引擎会据此判断前置知识是否掌握。

## 技术架构

```
前端：纯 HTML + CSS + JavaScript（静态页面，无需服务器）
题库：600+ 道结构化题目，全部标注知识点标签
存储：localStorage（每个学生独立模型，数据保存在本地浏览器）
部署：GitHub Pages / 任意静态托管
```
