# 四年级数学测试系统

一个专为北京海淀区四年级学生设计的数学测试生成与批改系统（基于北师大版教材）

## 功能特点

### 📝 1. 生成单元测试卷
- 支持8个单元的测试卷生成
- 三种难度等级选择
- 标准A4格式，适合打印
- 包含选择题、填空题、计算题、应用题
- 符合海淀区教学标准

### ✅ 2. 批改试卷与错题分析
- 自动批改学生答案
- 实时计算总分
- 详细的错题分析
- 按知识点归纳薄弱环节
- 为家长提供辅导建议

### 🎯 3. 生成针对性练习题
- 根据错题生成专项练习
- 10个知识点可选
- 3种题量选择（10/15/20题）
- 变式训练，巩固薄弱点

## 使用方法

### 在线使用
直接用浏览器打开 `index.html` 文件即可使用，无需安装任何软件。

### 使用步骤

#### 方法一：生成试卷
1. 点击"生成试卷"标签
2. 选择单元（第一单元到第八单元）
3. 选择难度（基础/中等/提高）
4. 点击"生成测试卷"按钮
5. 可以选择打印、下载或复制试卷

#### 方法二：批改试卷
1. 点击"批改试卷"标签
2. 输入学生姓名
3. 选择测试单元
4. 按照格式输入学生答案：
   ```
   1. D
   2. C
   6. 10, 100
   15. 3500-800=2700本，3500+2700=6200本
   ```
5. 点击"开始批改"按钮
6. 查看分数、错题分析和知识点归纳

#### 方法三：生成练习题
1. 点击"针对性练习"标签
2. 勾选需要练习的薄弱知识点
3. 选择题目数量
4. 点击"生成练习题"按钮
5. 可以打印或下载练习题

## 使用提示

### 批改答案格式要求
- 每行一题
- 格式：题号. 答案
- 选择题：`1. D`
- 填空题：`6. 10, 100`（多个空用逗号分隔）
- 计算题：`12. 5000`
- 应用题：`15. 3500-800=2700本，3500+2700=6200本`

### 打印设置建议
- 纸张大小：A4
- 方向：纵向
- 边距：默认
- 背景图形：关闭
